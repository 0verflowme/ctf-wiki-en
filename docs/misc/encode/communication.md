## Tap Code


Tap code is a simple way to encode the text. Tap code is based on a 5 Ã—5 square Polybius square, but the difference is that letter K is combined with the letter C.


|Tap Code|1|2|3|4|5|
|--|--|--|--|--|--|
|1|A|B|C/K|D|E|
|2|F|G|H|I|J|
|3|L|M|N|O|P|
|4|Q|R|S|T|U|
|5|V|W|X|Y|Z|

Example:

|Letter|F|O|X|
|--|--|--|--|
|Position|2,1|3,4|5,3|
|Tap Code|.. .|... ....|..... ...|
